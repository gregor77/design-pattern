# Awesome 디자인 패턴 시리즈
 1. Observer 패턴
    * 2015-05-20
 
* Observer 패턴 예
  * WOW 던전에 파티가 묶여 있는 것을 예를 들어 설명
    * 파티는 이 던전에 묶여있고, 보스몹 몇개 죽였는지 확인 가능
  * Java 버튼, 리스너
    * HeadFirst 샘플 예제 중에, SwingObserverExample.java (java 8 기준으로 작성되어 있음, 람다 표현식 사용)
    * HeadFirst 책 앞에 표기되어 있음
  * 공지사항 등록 후, 사용자에게 noti를 줄때
